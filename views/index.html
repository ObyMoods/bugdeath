JavaScript Obfuscator API

Enkripsi file JavaScript menjadi tidak terbaca (Memerlukan API Key - Limit berkurang)

JavaScript Obfuscator

Ready

Enkripsi file JavaScript (.js) menjadi kode yang sulit dibaca. Menggunakan konfigurasi strong obfuscation.

JSON READY
JAVASCRIPT
POST /api/js-obfuscate (multipart/form-data)
POST /api/js-obfuscate-text (application/json)
Perhatikan: Setiap obfuscation akan mengurangi LIMIT -1 dari limit Anda
Parameter (File Upload)
Parameter Deskripsi
file File JavaScript (.js) *wajib, max 5MB
apikey API Key Anda (via query) *wajib
Parameter (Text)
Parameter Deskripsi
code Kode JavaScript *wajib
apikey API Key Anda (via query) *wajib
Contoh Request (File Upload) 
const formData = new FormData();
formData.append('file', fileInput.files[0]);

fetch('http://pterodactyl.astavvip-tech.xyz:9389/api/js-obfuscate?apikey=YOUR_API_KEY', {
    method: 'POST',
    body: formData
})
.then(response => response.json())
.then(data => console.log(data));
Contoh Request (Text) 
fetch('http://pterodactyl.astavvip-tech.xyz:9389/api/js-obfuscate-text?apikey=YOUR_API_KEY', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ code: 'console.log("Hello World");' })
})
.then(response => response.json())
.then(data => console.log(data));
